Positions in CSS

Absolute

Position: absolute places an element relative to its parentâ€™s position and changing the layout around it.

Relative

Position: Relative places an element relative to its current position without changing the layout around it.

Fixed

Position: fixed mean fixed to the viewport. We tell it where to position itself (top, bottom, right, or left) and it will stay there when user scrolling.

Sticky

The element with position: sticky and top: 0 played a role between fixed & relative based on the position where it is placed.
